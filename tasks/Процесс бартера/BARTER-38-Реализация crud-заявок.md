# BARTER-38: Реализация crud-заявок

## Описание задачи
Реализация CRUD-операций для управления заявками в процессе бартера.

## Список действий
1. Создать модель заявки с необходимыми полями
2. Реализовать endpoint для создания заявки
3. Реализовать endpoint для чтения заявок
4. Реализовать endpoint для обновления заявки
5. Реализовать endpoint для удаления заявки
6. Реализовать валидацию данных и права доступа
7. Обеспечить связи с другими сущностями (пользователи, объявления)
8. Написать unit-тесты для всех операций
