# BARTER-42: Создание crud-сообщений

## Описание задачи
Реализация CRUD-операций для управления сообщениями в чатах.

## Список действий
1. Создать модель сообщения с необходимыми полями
2. Реализовать endpoint для создания сообщения
3. Реализовать endpoint для чтения сообщений
4. Реализовать endpoint для обновления сообщения
5. Реализовать endpoint для удаления сообщения
6. Реализовать валидацию данных и права доступа
7. Обеспечить защиту приватности сообщений
8. Написать unit-тесты для всех операций
