# BARTER-41: Создать чат обсуждения заявок

## Описание задачи
Реализация чата для обсуждения заявок между участниками бартера.

## Список действий
1. Создать модель чата для заявки
2. Реализовать создание чата при создании заявки
3. Обеспечить доступ к чату только участникам заявки
4. Реализовать endpoint для отправки сообщений в чат
5. Реализовать endpoint для получения сообщений из чата
6. Обеспечить получение сообщений в реальном времени
7. Обеспечить привязку чата к конкретной заявке
8. Протестировать функционал общения в чате
