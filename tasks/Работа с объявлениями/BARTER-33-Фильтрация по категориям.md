# BARTER-33: Фильтрация по категориям

## Описание задачи
Реализация фильтрации объявлений по категориям.

## Список действий
1. Определить структуру категорий для объявлений
2. Реализовать параметр фильтрации по категориям
3. Добавить поддержку иерархии категорий (подкатегории)
4. Создать интерфейс выбора категорий на фронтенде
5. Обеспечить валидацию выбранных категорий
6. Реализовать фильтрацию по нескольким категориям
7. Оптимизировать производительность запросов
8. Протестировать фильтрацию по всем категориям
