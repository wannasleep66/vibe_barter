# BARTER-28: Поиск объявлений по ключевым словам

## Описание задачи
Реализация поиска объявлений по ключевым словам в заголовке, описании и других полях.

## Список действий
1. Определить поля, по которым возможен поиск
2. Реализовать endpoint для поиска объявлений по ключевым словам
3. Настроить полнотекстовый поиск по релевантным полям
4. Обеспечить подсветку найденных слов в результатах
5. Реализовать сортировку результатов по релевантности
6. Оптимизировать производительность поиска
7. Добавить возможность комбинации поиска с другими фильтрами
8. Протестировать поиск с различными запросами
