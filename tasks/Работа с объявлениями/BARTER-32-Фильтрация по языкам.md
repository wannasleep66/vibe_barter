# BARTER-32: Фильтрация по языкам

## Описание задачи
Реализация фильтрации объявлений по языкам, указанным в профиле автора.

## Список действий
1. Определить структуру хранения языков в профиле пользователя
2. Реализовать параметр фильтрации по языкам
3. Добавить логику сопоставления языков автора и запроса
4. Создать интерфейс выбора языков для фильтрации
5. Обеспечить поддержку нескольких языков одновременно
6. Обновить документацию по доступным фильтрам
7. Оптимизировать производительность фильтрации
8. Протестировать фильтрацию по разным языкам
