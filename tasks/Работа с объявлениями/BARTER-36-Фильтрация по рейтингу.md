# BARTER-36: Фильтрация по рейтингу

## Описание задачи
Реализация фильтрации объявлений по рейтингу автора или самого объявления.

## Список действий
1. Определить, по какому рейтингу фильтровать (автора или объявления)
2. Реализовать параметр фильтрации по минимальному рейтингу
3. Добавить опционально фильтрацию по диапазону рейтинга
4. Создать интерфейс выбора диапазона рейтинга
5. Обеспечить актуальность рейтинга при фильтрации
6. Оптимизировать эффективность запросов с фильтрацией по рейтингу
7. Обновить документацию по фильтрам
8. Протестировать фильтрацию по разным значениям рейтинга
