# BARTER-23: Реализация crud-категорий

## Описание задачи
Реализация CRUD-операций для управления категориями объявлений.

## Список действий
1. Создать модель категории с необходимыми полями
2. Реализовать endpoint для создания категории
3. Реализовать endpoint для чтения категорий
4. Реализовать endpoint для обновления категории
5. Реализовать endpoint для удаления категории
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить поддержку иерархии категорий (родительские/дочерние)
8. Написать unit-тесты для всех операций
