# BARTER-24: Реализация crud-тегов

## Описание задачи
Реализация CRUD-операций для управления тегами объявлений.

## Список действий
1. Создать модель тега с необходимыми полями
2. Реализовать endpoint для создания тега
3. Реализовать endpoint для чтения тегов
4. Реализовать endpoint для обновления тега
5. Реализовать endpoint для удаления тега
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить уникальность тегов
8. Написать unit-тесты для всех операций
