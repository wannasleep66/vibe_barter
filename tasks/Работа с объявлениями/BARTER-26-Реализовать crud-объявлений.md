# BARTER-26: Реализовать crud-объявлений

## Описание задачи
Реализация CRUD-операций для управления объявлениями.

## Список действий
1. Создать модель объявления с необходимыми полями
2. Реализовать endpoint для создания объявления
3. Реализовать endpoint для чтения объявлений
4. Реализовать endpoint для обновления объявления
5. Реализовать endpoint для удаления объявления
6. Реализовать валидацию данных и права доступа
7. Обеспечить связи с другими сущностями (пользователи, теги, категории)
8. Написать unit-тесты для всех операций
