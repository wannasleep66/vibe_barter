# BARTER-44: Реализация crud-чатов

## Описание задачи
Реализация CRUD-операций для управления чатами между пользователями.

## Список действий
1. Создать модель чата с необходимыми полями
2. Реализовать endpoint для создания чата
3. Реализовать endpoint для чтения списка чатов
4. Реализовать endpoint для обновления информации о чате
5. Реализовать endpoint для удаления чата
6. Реализовать валидацию данных и права доступа
7. Обеспечить безопасность чата и приватность переписки
8. Написать unit-тесты для всех операций
