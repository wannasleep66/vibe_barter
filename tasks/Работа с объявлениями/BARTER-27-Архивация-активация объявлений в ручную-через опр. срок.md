# BARTER-27: Архивация/активация объявлений в ручную/через опр. срок

## Описание задачи
Реализация функций архивации и активации объявлений вручную или автоматически по истечении срока.

## Список действий
1. Добавить статус архивности в модель объявления
2. Реализовать endpoint для ручной архивации объявления
3. Реализовать endpoint для ручной активации объявления
4. Реализовать автоматическую архивацию по истечении срока
5. Создать интерфейс управления архивацией
6. Обеспечить уведомления о предстоящей архивации
7. Обновить логику отображения объявлений с учетом архивности
8. Протестировать оба метода архивации
