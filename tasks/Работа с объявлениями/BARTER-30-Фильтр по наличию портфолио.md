# BARTER-30: Фильтр по наличию портфолио

## Описание задачи
Реализация фильтрации объявлений по наличию у автора портфолио.

## Список действий
1. Определить критерии наличия портфолио у пользователя
2. Реализовать параметр фильтрации в endpoint для объявлений
3. Добавить логику проверки наличия портфолио у автора объявления
4. Создать интерфейс для фильтра на фронтенде
5. Обеспечить эффективность запроса с фильтром
6. Протестировать фильтрацию на разных данных
7. Обновить документацию по фильтрам
8. Проверить производительность с фильтром
