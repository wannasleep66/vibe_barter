# BARTER-25: Реализация crud-вложений_объявлений

## Описание задачи
Реализация CRUD-операций для управления вложениями (файлами) к объявлениям.

## Список действий
1. Создать модель вложений для объявлений
2. Реализовать endpoint для добавления вложения к объявлению
3. Реализовать endpoint для чтения вложений объявления
4. Реализовать endpoint для удаления вложения
5. Обеспечить ограничения на типы и размеры файлов
6. Реализовать безопасное хранение файлов
7. Обеспечить права доступа к вложениям
8. Написать unit-тесты для всех операций
