# BARTER-34: Фильтрация по тегам

## Описание задачи
Реализация фильтрации объявлений по тегам.

## Список действий
1. Определить структуру хранения тегов для объявлений
2. Реализовать параметр фильтрации по тегам
3. Добавить поддержку фильтрации по нескольким тегам
4. Создать интерфейс выбора тегов для фильтрации
5. Обеспечить валидацию выбранных тегов
6. Реализовать автодополнение при выборе тегов
7. Оптимизировать производительность фильтрации
8. Протестировать фильтрацию по разным комбинациям тегов
