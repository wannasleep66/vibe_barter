# BARTER-35: Фильтрация по срочности

## Описание задачи
Реализация фильтрации объявлений по срочности (срочное/обычное и т.п.).

## Список действий
1. Определить параметры срочности для объявлений
2. Добавить поле срочности в модель объявления
3. Реализовать параметр фильтрации по срочности
4. Создать интерфейс выбора уровня срочности
5. Обеспечить валидацию значений срочности
6. Реализовать сортировку по срочности
7. Обновить документацию по фильтрам
8. Протестировать фильтрацию по срочности
