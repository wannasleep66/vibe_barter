# BARTER-31: Фильтрация по локации

## Описание задачи
Реализация фильтрации объявлений по локации автора или месту проведения мероприятия.

## Список действий
1. Определить структуру хранения локации в профиле и объявлении
2. Реализовать параметр фильтрации по локации
3. Добавить поддержку географических координат (широта, долгота)
4. Реализовать фильтрацию по радиусу от заданной точки
5. Создать интерфейс выбора локации на фронтенде
6. Обеспечить валидацию вводимых локаций
7. Оптимизировать производительность запросов с геофильтрацией
8. Протестировать фильтрацию по разным локациям
