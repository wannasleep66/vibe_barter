# BARTER-10: Выход из аккаунта

## Описание задачи
Реализация функционала выхода из аккаунта пользователя.

## Список действий
1. Создать кнопку/ссылку для выхода на фронтенде
2. Реализовать endpoint для выхода из системы
3. Удалить JWT-токен на клиенте
4. Инвалидировать сессию на сервере при необходимости
5. Обработать внешние OAuth-сессии (при необходимости)
6. Перенаправить пользователя после выхода
7. Обеспечить безопасность процесса выхода
8. Протестировать функционал выхода
