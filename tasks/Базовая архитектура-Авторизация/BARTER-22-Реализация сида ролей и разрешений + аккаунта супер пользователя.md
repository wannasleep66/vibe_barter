# BARTER-22: Реализация сида ролей и разрешений + аккаунта супер пользователя

## Описание задачи
Реализация начального наполнения системы ролями и разрешениями, а также создание аккаунта супер пользователя.

## Список действий
1. Определить начальные роли и разрешения для системы
2. Реализовать скрипт/механизм сида ролей
3. Реализовать скрипт/механизм сида разрешений
4. Привязать разрешения к ролям
5. Создать аккаунт суперпользователя с полными правами
6. Обеспечить безопасность аккаунта суперпользователя
7. Автоматизировать процесс сида при развертывании
8. Протестировать заполнение начальными данными
