# BARTER-20: Создание crud-ролей

## Описание задачи
Реализация CRUD-операций для управления ролями.

## Список действий
1. Создать модель роли с необходимыми полями
2. Реализовать endpoint для создания роли
3. Реализовать endpoint для чтения информации о роли
4. Реализовать endpoint для обновления информации о роли
5. Реализовать endpoint для удаления роли
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только для администраторов)
8. Написать unit-тесты для всех операций
