# BARTER-7: Вход в аккаунт

## Описание задачи
Реализация функционала входа в аккаунт с использованием email и пароля.

## Список действий
1. Создать форму входа на фронтенде
2. Реализовать endpoint для аутентификации пользователя
3. Проверить правильность введенных креденциалов
4. Создать сессию для аутентифицированного пользователя
5. Реализовать валидацию данных на клиенте и сервере
6. Обработать случай неправильных креденциалов
7. Обеспечить защиту от атак подбора паролей
8. Протестировать функционал входа
