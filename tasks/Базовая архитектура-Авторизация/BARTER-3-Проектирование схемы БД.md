# BARTER-3: Проектирование схемы БД

## Описание задачи
Проектирование схемы базы данных для проекта.

## Список действий
1. Определить основные сущности системы
2. Спроектировать ER-диаграмму
3. Определить связи между сущностями
4. Спроектировать таблицы с нужными полями и типами данных
5. Определить индексы для оптимизации запросов
6. Спроектировать хранимые процедуры и триггеры при необходимости
7. Обеспечить целостность данных
8. Продумать стратегию миграций
