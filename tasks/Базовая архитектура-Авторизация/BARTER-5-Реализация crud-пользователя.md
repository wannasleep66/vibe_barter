# BARTER-5: Реализация crud-пользователя

## Описание задачи
Реализация операций CRUD (создание, чтение, обновление, удаление) для пользователей.

## Список действий
1. Создать модель пользователя с необходимыми полями
2. Реализовать endpoint для создания пользователя
3. Реализовать endpoint для чтения информации о пользователе
4. Реализовать endpoint для обновления информации о пользователе
5. Реализовать endpoint для удаления пользователя
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (авторизация)
8. Написать unit-тесты для всех операций
