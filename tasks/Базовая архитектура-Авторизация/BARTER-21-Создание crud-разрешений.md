# BARTER-21: Создание crud-разрешений

## Описание задачи
Реализация CRUD-операций для управления разрешениями.

## Список действий
1. Создать модель разрешения с необходимыми полями
2. Реализовать endpoint для создания разрешения
3. Реализовать endpoint для чтения информации о разрешении
4. Реализовать endpoint для обновления информации о разрешении
5. Реализовать endpoint для удаления разрешения
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только для администраторов)
8. Написать unit-тесты для всех операций
