# BARTER-19: Ролевая система

## Описание задачи
Реализация ролевой системы доступа для проекта.

## Список действий
1. Определить роли пользователей (администратор, модератор, обычный пользователь и т.д.)
2. Определить права доступа для каждой роли
3. Реализовать модель ролей и разрешений
4. Реализовать проверку ролей и разрешений в защищенных endpoint
5. Создать middleware для проверки ролей
6. Настроить права доступа к различным функциям системы
7. Реализовать интерфейс управления ролями
8. Протестировать ролевую систему
