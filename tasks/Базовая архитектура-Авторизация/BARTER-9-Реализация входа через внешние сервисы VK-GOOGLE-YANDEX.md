# BARTER-9: Реализация входа через внешние сервисы VK/GOOGLE/YANDEX

## Описание задачи
Реализация возможности входа через внешние OAuth-провайдеры (VK, Google, Yandex).

## Список действий
1. Настроить OAuth-приложения в VK, Google и Yandex
2. Выбрать библиотеку для OAuth аутентификации
3. Реализовать endpoint для инициации OAuth-процесса
4. Реализовать callback endpoint для обработки ответа от провайдеров
5. Реализовать привязку внешнего аккаунта к существующему или создание нового
6. Обработать случай, когда пользователь с email уже существует
7. Реализовать настройки OAuth в конфигурации приложения
8. Протестировать вход через каждый из провайдеров
