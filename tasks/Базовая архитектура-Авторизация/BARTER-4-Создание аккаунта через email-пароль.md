# BARTER-4: Создание аккаунта через email/пароль

## Описание задачи
Реализация функционала создания аккаунта с использованием email и пароля.

## Список действий
1. Создать форму регистрации на фронтенде
2. Реализовать валидацию данных на клиенте и сервере
3. Реализовать хеширование паролей
4. Создать endpoint для регистрации пользователя
5. Реализовать проверку уникальности email
6. Обеспечить безопасность паролей (длина, сложность)
7. Сохранить пользователя в базе данных
8. Протестировать функционал регистрации
