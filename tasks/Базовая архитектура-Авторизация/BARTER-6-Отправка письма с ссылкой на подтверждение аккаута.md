# BARTER-6: Отправка письма с ссылкой на подтверждение аккаута

## Описание задачи
Реализация отправки письма с ссылкой для подтверждения аккаунта зарегистрированному пользователю.

## Список действий
1. Настроить систему отправки email (SMTP)
2. Создать шаблон письма с подтверждением аккаунта
3. Реализовать генерацию уникальной ссылки подтверждения
4. Добавить поле подтверждения email в модель пользователя
5. Реализовать endpoint для обработки подтверждения по ссылке
6. Обеспечить безопасность токена подтверждения (время жизни, одноразовость)
7. Отправить письмо при регистрации пользователя
8. Протестировать процесс подтверждения
