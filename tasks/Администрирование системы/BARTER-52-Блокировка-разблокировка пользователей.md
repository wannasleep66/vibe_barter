# BARTER-52: Блокировка/разблокировка пользователей

## Описание задачи
Реализация функций блокировки и разблокировки пользователей администратором.

## Список действий
1. Добавить статус блокировки в модель пользователя
2. Реализовать endpoint для блокировки пользователя
3. Реализовать endpoint для разблокировки пользователя
4. Создать интерфейс управления блокировками в админ-панели
5. Обеспечить уведомление пользователя о блокировке
6. Реализовать причину блокировки (необязательно)
7. Обработать доступ к системе заблокированными пользователями
8. Протестировать процессы блокировки и разблокировки
