# BARTER-53: Назначение / отзыв ролей

## Описание задачи
Реализация функций назначения и отзыва ролей у пользователей администратором.

## Список действий
1. Реализовать endpoint для назначения роли пользователю
2. Реализовать endpoint для отзыва роли у пользователя
3. Создать интерфейс управления ролями в админ-панели
4. Обеспечить проверку корректности назначаемой роли
5. Реализовать логирование изменений ролей
6. Обеспечить защиту от неправомерного назначения ролей
7. Обновить права доступа пользователя при изменении роли
8. Протестировать процесс назначения и отзыва ролей
