# BARTER-51: Фильтрация + поиск пользователей

## Описание задачи
Реализация функций фильтрации и поиска пользователей в административном интерфейсе.

## Список действий
1. Определить поля, по которым возможна фильтрация пользователей
2. Реализовать фильтрацию по статусу, ролям, дате регистрации и другим параметрам
3. Реализовать поиск по основным полям (имя, email, ID)
4. Настроить сортировку результатов поиска
5. Создать интерфейс фильтрации на фронтенде
6. Обеспечить производительность при поиске по большим базам
7. Реализовать комбинации фильтров
8. Протестировать функции поиска и фильтрации
