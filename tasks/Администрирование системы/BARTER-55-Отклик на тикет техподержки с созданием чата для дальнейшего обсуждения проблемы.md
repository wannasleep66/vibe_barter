# BARTER-55: Отклик на тикет техподержки с созданием чата для дальнейшего обсуждения проблемы

## Описание задачи
Реализация функции отклика на тикет техподдержки с созданием чата для обсуждения проблемы.

## Список действий
1. Реализовать функцию отклика на тикет для агентов поддержки
2. Создать автоматическое создание чата при отклике на тикет
3. Привязать чат к соответствующему тикету
4. Обеспечить доступ к чату для пользователя и агента поддержки
5. Реализовать уведомления о создании чата
6. Обновить статус тикета при создании чата
7. Обеспечить архивацию чата при закрытии тикета
8. Протестировать процесс отклика и создания чата
