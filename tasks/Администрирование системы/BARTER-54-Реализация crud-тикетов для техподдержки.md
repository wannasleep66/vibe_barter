# BARTER-54: Реализация crud-тикетов для техподдержки

## Описание задачи
Реализация CRUD-операций для управления тикетами технической поддержки.

## Список действий
1. Создать модель тикета с необходимыми полями
2. Реализовать endpoint для создания тикета
3. Реализовать endpoint для чтения тикетов
4. Реализовать endpoint для обновления тикета
5. Реализовать endpoint для удаления тикета
6. Реализовать валидацию данных и права доступа
7. Обеспечить уведомления о новых тикетах для поддержки
8. Написать unit-тесты для всех операций
