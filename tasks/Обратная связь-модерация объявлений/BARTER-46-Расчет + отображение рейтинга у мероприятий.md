# BARTER-46: Расчет + отображение рейтинга у мероприятий

## Описание задачи
Реализация расчета и отображения рейтинга у мероприятий на основе отзывов.

## Список действий
1. Определить алгоритм расчета рейтинга на основе отзывов
2. Реализовать вычисление среднего рейтинга для мероприятий
3. Реализовать обновление рейтинга при добавлении/изменении отзывов
4. Создать endpoint для получения рейтинга мероприятия
5. Реализовать отображение рейтинга на фронтенде
6. Обеспечить актуальность рейтинга в реальном времени
7. Протестировать корректность расчетов
8. Документировать алгоритм расчета рейтинга
