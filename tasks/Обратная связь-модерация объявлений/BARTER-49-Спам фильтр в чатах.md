# BARTER-49: Спам фильтр в чатах

## Описание задачи
Реализация фильтра спама в чатах между пользователями.

## Список действий
1. Определить критерии спама в сообщениях
2. Реализовать систему анализа сообщений на спам
3. Создать блэклист слов/фраз для фильтрации
4. Реализовать автоматическую блокировку спам-сообщений
5. Обеспечить возможность ручной проверки спорных сообщений
6. Реализовать уведомления о блокировке сообщений
7. Создать статистику по заблокированным сообщениям
8. Протестировать эффективность фильтра
