# BARTER-48: Сокрытие неприемлимых объявлений

## Описание задачи
Реализация функционала для скрытия неприемлемых объявлений (модерация).

## Список действий
1. Определить критерии неприемлемого содержания
2. Реализовать механизм отмечания объявлений как неприемлемых
3. Реализовать автоматическую систему фильтрации
4. Создать интерфейс для модераторов по проверке объявлений
5. Реализовать скрытие объявлений на фронтенде
6. Обеспечить уведомление владельца объявления о скрытии
7. Реализовать возможность обжалования скрытия
8. Протестировать систему модерации
