# BARTER-45: Реализация crud-отзывов

## Описание задачи
Реализация CRUD-операций для управления отзывами пользователей.

## Список действий
1. Создать модель отзывов с необходимыми полями
2. Реализовать endpoint для создания отзыва
3. Реализовать endpoint для чтения отзывов
4. Реализовать endpoint для обновления отзыва
5. Реализовать endpoint для удаления отзыва
6. Реализовать валидацию данных и проверку прав доступа
7. Обеспечить ограничения (например, один отзыв на взаимодействие)
8. Написать unit-тесты для всех операций
