# BARTER-47: Фильтр стоп-слов при создании объявлений

## Описание задачи
Реализация фильтра стоп-слов при создании объявлений для предотвращения нежелательного контента.

## Список действий
1. Создать список стоп-слов для фильтрации
2. Реализовать проверку текста объявления на наличие стоп-слов
3. Реализовать валидацию при создании и обновлении объявления
4. Заблокировать создание объявления с запрещенным контентом
5. Уведомить пользователя о наличии стоп-слов в тексте
6. Обеспечить гибкость списка стоп-слов (возможность обновления)
7. Реализовать частичное совпадение и вариации слов
8. Протестировать фильтр на различных сценариях
