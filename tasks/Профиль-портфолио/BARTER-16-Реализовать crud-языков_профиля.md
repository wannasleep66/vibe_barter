# BARTER-16: Реализовать crud-языков_профиля

## Описание задачи
Реализация CRUD-операций для управления языками в профиле пользователя.

## Список действий
1. Создать модель языков профиля с необходимыми полями
2. Реализовать endpoint для добавления языка к профилю
3. Реализовать endpoint для чтения списка языков профиля
4. Реализовать endpoint для обновления информации о языке
5. Реализовать endpoint для удаления языка из профиля
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только владелец или админ)
8. Написать unit-тесты для всех операций
