# BARTER-18: Реализовать crud-портфолио

## Описание задачи
Реализация CRUD-операций для управления портфолио пользователя.

## Список действий
1. Создать модель портфолио с необходимыми полями
2. Реализовать endpoint для создания портфолио
3. Реализовать endpoint для чтения информации о портфолио
4. Реализовать endpoint для обновления информации о портфолио
5. Реализовать endpoint для удаления портфолио
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только владелец или админ)
8. Написать unit-тесты для всех операций
