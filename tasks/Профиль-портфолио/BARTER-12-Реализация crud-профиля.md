# BARTER-12: Реализация crud-профиля

## Описание задачи
Реализация операций CRUD для управления профилем пользователя.

## Список действий
1. Создать модель профиля с необходимыми полями
2. Реализовать endpoint для создания профиля
3. Реализовать endpoint для чтения информации профиля
4. Реализовать endpoint для обновления информации профиля
5. Реализовать endpoint для удаления профиля
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только владелец или админ)
8. Написать unit-тесты для всех операций
