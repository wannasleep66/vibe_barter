# BARTER-15: Реализовать crud-навыков_профиля

## Описание задачи
Реализация CRUD-операций для управления навыками в профиле пользователя.

## Список действий
1. Создать модель навыков профиля с необходимыми полями
2. Реализовать endpoint для добавления навыка к профилю
3. Реализовать endpoint для чтения списка навыков профиля
4. Реализовать endpoint для обновления информации о навыке
5. Реализовать endpoint для удаления навыка из профиля
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только владелец или админ)
8. Написать unit-тесты для всех операций
