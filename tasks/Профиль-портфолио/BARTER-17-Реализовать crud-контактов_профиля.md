# BARTER-17: Реализовать crud-контактов_профиля

## Описание задачи
Реализация CRUD-операций для управления контактами в профиле пользователя.

## Список действий
1. Создать модель контактов профиля с необходимыми полями
2. Реализовать endpoint для добавления контакта к профилю
3. Реализовать endpoint для чтения списка контактов профиля
4. Реализовать endpoint для обновления информации о контакте
5. Реализовать endpoint для удаления контакта из профиля
6. Реализовать валидацию данных при создании/обновлении
7. Обеспечить права доступа к операциям (только владелец или админ)
8. Написать unit-тесты для всех операций
